# 📂 Revisão 05: Filtros Inteligentes e Atalhos

Este diretório contém os exercícios práticos focados em otimizar as buscas no banco de dados usando atalhos avançados do `WHERE`, baseados no curso SQL Impressionador (Hashtag Treinamentos).

## 🧠 O que eu aprendi neste tópico
* **O Buscador `LIKE`**: Como buscar por fragmentos de texto dentro de uma coluna usando o curinga `%` (ex: encontrar qualquer produto que contenha a palavra 'Gamer' no nome).
* **O Atalho `IN` e `NOT IN`**: Como substituir múltiplos comandos `OR` por uma lista limpa e performática, permitindo incluir ou excluir múltiplos valores de uma só vez.
* **O Comando `BETWEEN`**: A forma mais elegante de filtrar intervalos (sejam números ou datas), simplificando a leitura do código em relação ao uso de `>=` e `<=`.
