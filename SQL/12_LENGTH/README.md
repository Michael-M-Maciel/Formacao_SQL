# 📂 Revisão 12: Medindo Dados (Função LENGTH)

Este diretório registra o aprendizado sobre a função de mensuração de caracteres, essencial para validação de integridade e qualidade dos dados.

## 🧠 O que aprendi neste tópico
* **Contagem de Caracteres**: Como utilizar o `LENGTH()` para medir o tamanho de strings, incluindo espaços e caracteres especiais.
* **Validação de Qualidade**: Aplicação da função para identificar dados incompletos ou fora do padrão esperado (ex: e-mails muito curtos ou IDs fora de conformidade).
* **Filtros por Comprimento**: Uso de condições lógicas no `WHERE` baseadas no tamanho do texto para segmentar categorias (ex: marcas com nomes curtos vs. nomes longos).
