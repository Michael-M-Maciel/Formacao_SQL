# 📂 Revisão 10: User-Defined Variables (Variáveis de Usuário)

Este diretório contém o registro do meu aprendizado sobre o uso de variáveis dinâmicas no MySQL, uma técnica essencial para automação de scripts e simulações financeiras.

## 🧠 O que aprendi neste tópico
* **Declaração com `SET`**: Como reservar um espaço na memória do banco de dados para armazenar valores (números ou textos) usando o prefixo `@`.
* **Filtros Dinâmicos**: Utilização de variáveis em cláusulas `WHERE` para facilitar a alteração de parâmetros (como metas ou IDs de clientes) sem modificar a estrutura da query.
* **Cálculos Reutilizáveis**: Aplicação de variáveis em operações matemáticas, permitindo simular cenários de impostos, descontos e reajustes de preços de forma centralizada.
