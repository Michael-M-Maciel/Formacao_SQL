# 📂 Revisão 09: Contagens e Unicidade (COUNT e COUNT DISTINCT)

Este diretório contém o registro do meu aprendizado sobre funções de contagem no MySQL, essenciais para o início de qualquer análise exploratória de dados.

## 🧠 O que aprendi neste tópico
* **Função `COUNT(*)`**: Utilizada para mensurar o volume total de registros em uma tabela, ignorando se os dados se repetem.
* **Função `COUNT(DISTINCT column)`**: Fundamental para identificar a abrangência e a variedade de uma base, contando apenas os valores exclusivos (únicos).
* **Filtros com Contagem**: Aplicação de cláusulas `WHERE` em conjunto com funções de agregação para obter métricas específicas de performance (ex: lojas com vendas ativas).
