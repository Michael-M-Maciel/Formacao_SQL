# 📂 Revisão 04: Filtros Avançados e Lógica Booleana

Este diretório contém os exercícios práticos focados no comando `WHERE` e nos operadores lógicos do MySQL, baseados no curso SQL Impressionador (Hashtag Treinamentos).

## 🧠 O que eu aprendi neste tópico
* **O Comando `WHERE`**: Como filtrar dados linha por linha com base em condições matemáticas (`<`, `>`, `=`).
* **Operadores Lógicos (`AND`, `OR`)**: Como combinar múltiplas condições para criar filtros extremamente precisos (exigindo que ambas as condições sejam verdadeiras ou aceitando alternativas).
* **Ausência de Dados (`IS NULL` vs `Vazio`)**: A diferença crítica entre um campo que nunca foi preenchido (`NULL`) e um campo preenchido com um texto em branco (`''`), e como filtrar corretamente ambas as situações.
