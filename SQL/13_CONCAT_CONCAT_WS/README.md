# 📂 Revisão 13: Junção de Textos (CONCAT e CONCAT_WS)

Este diretório registra o aprendizado sobre funções de manipulação de strings para consolidação de informações e formatação de relatórios.

## 🧠 O que aprendi neste tópico
* **Função `CONCAT`**: Utilizada para unir duas ou mais strings. Exige a inserção manual de separadores (espaços, hifens, etc) entre as colunas.
* **Função `CONCAT_WS` (With Separator)**: Uma forma mais eficiente de unir múltiplos campos, onde definimos um separador fixo apenas uma vez no início da função.
* **Criação de Identificadores**: Como gerar códigos de rastreio e etiquetas descritivas combinando IDs numéricos com textos fixos.
* **Personalização de Mensagens**: Formatação de frases dinâmicas para comunicação ou auditoria de dados.
