# 📂 Revisão 06: Funções de Agregação e Matemática Básica

Este diretório contém os exercícios práticos focados na geração de métricas e indicadores de negócios no MySQL, baseados no curso SQL Impressionador (Hashtag Treinamentos).

## 🧠 O que eu aprendi neste tópico
* **O Comando `COUNT`**: A diferença fundamental entre contar o total de registros de uma tabela (`COUNT(*)`) e contar apenas os registros únicos e exclusivos de uma coluna (`COUNT(DISTINCT)`).
* **Métricas Financeiras (`SUM` e `AVG`)**: Como utilizar o banco de dados para somar faturamentos totais ou descobrir o ticket médio/média de valores sem precisar exportar os dados para o Excel.
* **Extremos (`MIN` e `MAX`)**: Como identificar rapidamente os maiores e menores valores (ex: produto mais caro, cliente com maior renda) dentro de uma base com milhares de linhas.
