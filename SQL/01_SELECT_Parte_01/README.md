# 📂 Revisão 01: O Comando SELECT (Parte 1)

Este diretório contém os meus resumos e exercícios práticos sobre o comando básico de extração de dados no MySQL, baseados no curso SQL Impressionador (Hashtag Treinamentos).

## 🧠 O que eu aprendi neste tópico
* **Varredura Completa (`SELECT *`)**: Como utilizar o asterisco para trazer todas as colunas e registros de uma tabela rapidamente, ideal para reconhecer a estrutura dos dados.
* **Seleção Específica (`SELECT coluna1, coluna2`)**: Como especificar exatamente as colunas desejadas para gerar relatórios mais limpos, focados e otimizar o processamento do banco de dados.
* **Boas Práticas**: A importância da sintaxe correta e de sempre encerrar os comandos SQL com o ponto e vírgula (`;`).
