# 📂 Revisão 03: Ordenação, Eliminação de Duplicatas e Paginação

Este diretório contém exercícios focados na organização visual e estrutural dos relatórios no MySQL, baseados no curso SQL Impressionador (Hashtag Treinamentos).

## 🧠 O que eu aprendi neste tópico
* **`ORDER BY`**: Como ordenar dados de forma crescente (padrão) ou decrescente (`DESC`), essencial para criar rankings ou listas alfabéticas.
* **`DISTINCT`**: A forma mais otimizada de eliminar dados duplicados de um relatório, garantindo que valores repetidos apareçam apenas uma vez, sem a necessidade de usar funções de agrupamento complexas.
* **`LIMIT` e `OFFSET`**: Como controlar a quantidade de linhas retornadas. O uso combinado dessas cláusulas é a base de como a paginação (Página 1, Página 2) funciona nos sites modernos.
