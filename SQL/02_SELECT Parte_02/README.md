# 📂 Revisão 02: SELECT (Parte 2) e Comando AS

Este diretório contém os meus resumos e exercícios práticos sobre operações matemáticas no momento da extração de dados e a renomeação de colunas no MySQL, baseados no curso SQL Impressionador (Hashtag Treinamentos).

## 🧠 O que eu aprendi neste tópico
* **Operações Matemáticas (`+, -, *, /`)**: Como utilizar o SGBD como uma calculadora para criar novas colunas de análise em tempo real (ex: calcular lucros, descontos ou taxas), sem alterar os dados originais da tabela.
* **Comando Alias (`AS`)**: A importância de renomear colunas no resultado da pesquisa (`SELECT coluna AS 'Novo_Nome'`) para gerar relatórios mais limpos, legíveis e com aparência profissional para o utilizador final.
