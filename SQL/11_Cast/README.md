# 📂 Revisão 11: Conversão de Dados (Função CAST)

Este repositório registra o meu aprendizado sobre a tipagem de dados no MySQL e como realizar conversões seguras entre textos, números e datas.

## 🧠 O que aprendi neste tópico
* **Tipagem Dinâmica**: A importância de garantir que o dado esteja no formato correto para operações matemáticas (ex: somar textos que representam números).
* **Tipos de Conversão**:
  * `DECIMAL`: Para valores financeiros precisos.
  * `SIGNED`: Para converter valores para números inteiros (positivos ou negativos).
  * `DATE`: Para transformar strings de texto em objetos de data válidos.
  * `CHAR`: Para converter números em texto, facilitando a concatenação futura.
* **Limpeza de Dados**: Como simplificar visualizações removendo casas decimais indesejadas em relatórios de grande escala.
